请求参数1： params = {"username"} 表示请求中必须含有关建字（key）username，对属性值（value）不做要求
请求参数2： params = {"!username"} 表示请求中不能含有关建字（key）username
请求参数3： params = {"username=admin"} 表示请求中必须含有关建字（key）username，且属性值（value）必须为admin
请求参数4： params = {"username!=admin"} 表示请求中必须含有关建字（key）username，且属性值（value）不能为为admin
请求参数1： params/headers = {"username"}
请求参数2： params/headers = {"!username"}
请求参数3： params/headers = {"username=admin"}
请求参数4： params/headers = {"username!=admin"}
?表示任意单个字符
*表示任意的0个或多个字符
**表示任意的一层或多层目录
